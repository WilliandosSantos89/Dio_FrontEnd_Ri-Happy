Seletores CSS

O CSS possui vários tipos de seletores

  • Seletor por Tipo/tag: Busca elementos por uma tag HTML específica
  • Seletor por ID (#): Busca elementos pelo atributo ID
  • Seletor por Classes (.): Busca elementos através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos
  • Seletor por Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nsso documento HTML e conseguimos também buscar atributos com um valor específico.
