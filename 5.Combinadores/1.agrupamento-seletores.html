Agrupamento de Seletores

Podemos aplicar as mesmas regraas CSS para vários seletores diferentes. Para isso, podemos separar os seletores usando uma vírgula.

Assim, não precisamos ficar copindo e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras.
